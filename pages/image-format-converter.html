JPG vs PNG - Which Format to Use?

JPG Format: Best for photographs and images with many colors. Smaller file size but doesn't support transparency. Most exam portals accept JPG format.
PNG Format: Lossless compression with transparency support. Slightly larger file size but maintains perfect quality. Required by some exam applications.
WEBP Format: Modern format with excellent compression. Converts easily to JPG or PNG when needed.
